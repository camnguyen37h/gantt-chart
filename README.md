# Project Management Dashboard

A React-based project management dashboard built with Ant Design v3.

## Features

- Project Overview with Gantt Chart
- Project Schedule Timeline
- Workforce Planning Analytics with Bar Charts
- KPI Metrics Dashboard
- Responsive Design

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Tech Stack

- React 16.14 (Function Components + Hooks)
- Ant Design 3.26.20
- Recharts 2.x
- React Router 5.x
- Moment.js
